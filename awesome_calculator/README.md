This calculator has been tested in "Medium Phone API 36.0 (anroid-x64 emulator) and Windows (windows-desktop).

Some hints:

Type `PLOT(SIN, 0, 2*PI)` <enter> to plot a SINE.
Type `PLOT((x)=>x^2, 0, 10)` <enter> to plot `x^2`.
Press `LOAD` and select `calculator.shql` to run a calculator program.
If a pogram hangs, press `BREAK` to abort execution.
Type `CLS` <enter> to remove text from the screen or `HIDE_GRAPH` to hide the graph from the plot fuction.
